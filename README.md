## How to run project

Clone the project

```bash

git clone https://github.com/mohammadyousefvand/Css-Generator-React.git

```

Go to the project directory

```bash

cd Css-Generator-React

```

Install dependencies

```bash

npm install

```

Start the server

```bash

npm start

```
